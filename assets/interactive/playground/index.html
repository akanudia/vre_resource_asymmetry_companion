What's this?

What you're looking at. The 4×2 chart matrix shows every paired-shift point from the 1,080-instance scenario ensemble for the currently selected region. Columns are outcomes (cumulative wind, solar, system cost, CO2 emissions). Rows are structural channels: supply-curve refinement (LoT → HiT) and temporal-resolution refinement (TS04 → TS72).

How to read a point. x = the parametric % shift of the coarse leg of the pair vs the C7-Base-median anchor. y = the structural % shift coarse → fine, on the same denominator. A point outside the dashed y = ±x wedge is structurally dominant — the resolution choice moved the outcome by more than the parametric distance from the anchor.

Using the controls. Pick a different region from the combo. Recolour or re-marker the cloud by any scenario axis. Filter chips narrow the cloud — checked = included; unchecked chips fade out so it's visible what you've turned off. Click any chip-group label (e.g. CLIMATE) to toggle all its chips on or off. The current view is encoded in the page URL; copy-paste the URL to share an exact view.
